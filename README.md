# Sales Case Study -Promotion Impact & Price Elasticity
**BrightLearn Data Analytics ** 

## Key Fingings 
- Average Unit Price : **R1428.38**
- Promotions on **57.7%** of days
- Volume lift during promo : **+41.8%**
- Margin drops to **-2.1%** (vs +29.4% normal)
- Price Elasticity = ** -1 .8** → **Highly elastic**

  Recommendation:**Run short ,targeted promotions only (<30% of days).
